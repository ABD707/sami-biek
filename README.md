# مطعم سامي بيك - النظام المحاسبي والإداري 🍗

نظام محاسبي وإداري متكامل لمطعم **سامي بيك** في حدائق أكتوبر - مصر.
يعمل بالكامل في المتصفح بدون أي خادم. كل البيانات محفوظة محلياً في `LocalStorage`.

## 🎯 الميزات

- **لوحة تحكم ذكية** مع رسوم بيانية + KPIs
- **التارجت المطلوب**: يحسب تلقائياً المبيعات اللازمة لتغطية كل التكاليف (إيجار + رواتب + كهرباء + غاز + خامات)
- **نقطة بيع (POS)** للفواتير اليومية - دفع كاش أو فيزا طلبات
- **إدارة المخزون** مع تنبيهات الحد الأدنى ونفاد الخامات
- **المشتريات** تُحدث المخزون تلقائياً بمتوسط السعر المرجح
- **الموظفون والرواتب** مع سجل المدفوعات
- **المصاريف الثابتة** (إيجار، كهرباء، غاز، مياه، إنترنت...)
- **تقارير ذكية** يومية / أسبوعية / شهرية / سنوية
- **نصائح ذكية** تلقائية لتحسين الأداء واكتشاف الخسائر
- **نسخ احتياطية JSON** للتصدير والاستيراد
- واجهة عربية كاملة (RTL) - تعمل على الموبايل والكمبيوتر

## 🚀 التشغيل المحلي

افتح ملف `index.html` بأي متصفح حديث (Chrome / Edge / Safari).

## 🌐 الرفع على GitHub Pages

1. أنشئ مستودع جديد (Repository) على GitHub
2. ارفع كل الملفات للمستودع
3. من إعدادات المستودع: `Settings → Pages → Source = main branch → / (root)`
4. الموقع سيشتغل على رابط: `https://<username>.github.io/<repo-name>/`

## 📂 الملفات

- `index.html` - النظام الرئيسي (الذي يفتحه GitHub Pages)
- `samy.html` - نسخة احتياطية مطابقة لـ index.html
